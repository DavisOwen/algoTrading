class performance.PerformanceHandler(results_dir)¶

Object for saving and loading backtest results and log files. Includes functions to analyze results such as plot_equity_curve

_create_drawdowns(equity_curve)¶

Calculate the largest peak-to-trough drawdown of the PnL curve as well as the duration of the drawdown. Requires that the pnl_returs is a pandas Series.

Parameters

equity_curve (pd.Series) – a pandas Series representing period percentage returns.

Returns

drawdown, duration - highest peak-to-trough drawdown and duration

Return type

float, float

_create_sharpe_ratio(returns, periods=252)¶

Create the Sharpe ratio for the strategy, based on a benchmark of zero (i.e. no risk-free rate information).

Parameters

  • returns (pd.Series) – a pandas Series representing period percentage returns

  • periods (int) – Daily (252), Hourly (252*6.5), Minutely(252*6.5*60) etc

Returns

sharpe ratio

Return type

float
