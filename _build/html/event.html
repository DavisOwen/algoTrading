event¶

class event.Event¶

Event is base class providing an interface for all subsequent (inhereted) events, that will trigger further events in the trading infrastructure.

class event.FillEvent(timeindex, symbol, exchange, quantity, direction, fill_cost, commission=None)¶

Encapsulates the notion of a Filled Order, as returned from a brokerage. Stores the quantity of an instrument actually filled and at what price. In addition, stores the comission of the trade from the brokerage.

calculate_ib_commission()¶

Calculate the fees of trading based on an Interactive Brokers fee structure for API, in USD.

This does not include exchange or ECN fees.

Based on “US API Directed Orders”: https://www.interactivebrokers.com/en/index.php?f=commission&p=stocks2

Returns

simulated ib commission cost

Return type

float

class event.MarketEvent¶

Handles the event of receiving a new market update with corresponding bars.

class event.OrderEvent(symbol, order_type, quantity, direction)¶

Handles the event of sending an Order to an execution system. The order contains a symbol (e.g. GOOG), a type (market or limit), quantity and a direction.

print_order()¶

Outputs the values within the Order.

class event.SignalEvent(symbol, datetime, signal_type, strength=None)¶

Handles the event of sending a Signal from a Strategy object. This is received by a Portfolio object and acted upon.
